# Общий словарь терминов — все системы

Термины, общие для всех TTRPG-систем в проекте. Минимальный набор — только то, что переводится одинаково везде.

Формат: `Оригинал | Перевод | Комментарий`

---

| Оригинал | Перевод | Комментарий |
|---|---|---|
| Attack | Атака | |
| Campaign | Кампания | |
| Cover | Укрытие | |
| Damage | Урон | |
| Game Master (GM) | Мастер игры (МИ) | Не "Ведущий", не "ГМ". Исключение: если система имеет своё название |
| Hit Points (HP) | Хиты (ХП) | |
| Non-Player Character (NPC) | Неигровой персонаж (НИП) | |
| Resistance | Сопротивление | |
| Session | Сессия | |
