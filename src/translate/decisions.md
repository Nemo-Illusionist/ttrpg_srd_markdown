# Терминологические решения — общие (все системы)

Решения, применимые ко всем TTRPG-системам в проекте.

---

### Gamemaster → Мастер игры (не "Ведущий", не "ГМ")

- **Причина:** Наиболее распространённый термин в русскоязычном TTRPG-сообществе
- **Исключение:** Если в системе есть специальное название для GM (например, Storyteller в WoD) — использовать его

### Dice notation: без перевода

- D20, 2D6, D100, 1D8+1 — остаются как есть
- **Причина:** Универсальная нотация, понятная всем игрокам

### Feet → футов (не "клеток", не "метров")

- **Причина:** SRD использует футы как единицу измерения. Конвертация в метры изменила бы баланс
- **Отвергнуто:** "Клетки" (не всегда применимо), "Метры" (другой масштаб)
