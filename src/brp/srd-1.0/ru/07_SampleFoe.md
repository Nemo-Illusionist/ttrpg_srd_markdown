# 7. Пример противника

Следующее существо представлено как ориентир для Ведущего при описании Неигровых персонажей (НИП), монстров и других противников. Как правило, нечеловеческие противники не имеют значения ВНШ.

## Медведь

Крупные наземные хищники, медведи — четвероногие, покрытые густым мехом когтистые всеядные. Большинство медведей обитает в лесах, а белые медведи живут в арктических регионах. Обычно они свирепы, защищая своё потомство. Здесь представлен чёрный медведь — один из самых распространённых видов.

| Характеристика | Бросок | Среднее |
|---|---|---|
| СИЛ | 3D6+10 | 20–21 |
| ТЕЛ | 2D6+6 | 13 |
| РАЗ | 3D6+10 | 20–21 |
| ИНТ | 5 | 5 |
| ВОЛ | 3D6 | 10–11 |
| ЛОВ | 3D6 | 10–11 |

**Перемещение:** 14 (8 плавание)

**Хиты:** 17

**Бонус урона:** +2D6

**Броня:** мех 3 очка

**Навыки:** Лазание 40%, Слух 75%, Чутьё 75%.

### Атаки

| Тип | % | Урон |
|---|---|---|
| Укус\* | 25 | 1D10+1/2 бонус урона (бу) |
| Когти\* | 40 | 1D6+бу |
| Удар лапой\*\* | 25 | 1D3+бу |

\* Медведь может атаковать дважды за боевой раунд, используя либо две отдельные атаки когтями, либо одну атаку когтями и одну атаку укусом.

\*\* Медведь может совершить одну атаку ударом лапой за боевой раунд, сопоставляя нанесённый медведем урон с СИЛ цели по Таблице сопротивления. При провале цель сбита с ног. Они могут парировать.
