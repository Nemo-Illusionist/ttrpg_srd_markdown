# Справочник навыков

Все 56 навыков из SRD Basic Roleplaying, отсортированные по алфавиту с базовыми шансами и описаниями.

## Категории навыков

- **Боевые навыки** -- непосредственный бой и использование оружия
- **Ментальные навыки** -- знания, рассуждения и восприятие
- **Навыки восприятия** -- осведомлённость и обнаружение
- **Навыки манипуляции** -- тонкие физические задачи и ремёсла
- **Навыки общения** -- социальное взаимодействие и убеждение
- **Физические навыки** -- атлетические и телесные действия

## Улучшение навыков

Навыки улучшаются через **Отметки опыта**:

1. После успешного использования навыка во время сессии игрок ставит отметку опыта для этого навыка.
2. Между сессиями игрок бросает D100 для каждого отмеченного навыка.
3. Если результат броска **выше** текущего значения навыка, навык повышается на 1D6 пунктов.
4. Навыки выше 100% всё ещё могут повышаться, но делать это становится сложнее.

## Модификаторы навыков

| Модификатор | Оригинал | Эффект |
|---|---|---|
| Дополняющий навык | Complementary Skill | Успех по связанному навыку может дать бонус |
| Лёгкий (x2) | Easy (x2) | Удвоение шанса навыка для простых задач |
| Противоборствующий бросок | Opposed Roll | Оба участника бросают; побеждает меньший успешный результат |
| Сложный (/2) | Difficult (/2) | Половинный шанс навыка для трудных задач |
