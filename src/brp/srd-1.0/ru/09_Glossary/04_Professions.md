# Справочник профессий

Все 12 профессий из SRD Basic Roleplaying с полными списками навыков, ключевыми характеристиками и описаниями.

## Как работают профессии

Профессия определяет, каким навыкам персонаж может распределить **профессиональные очки навыков** при создании персонажа. Профессиональные очки навыков равны сумме двух основных характеристик профессии, умноженной на зависящий от сеттинга множитель (обычно x10 или x20).

Персонажи также получают **очки личных интересов** (обычно ИНТ x10) для свободного распределения по любым навыкам.

## Краткое руководство по созданию персонажа

1. **Бросить характеристики** (СИЛ, ТЕЛ, РАЗ, ИНТ, МОЩ, ЛОВ, ВНШ)
2. **Выбрать профессию** из списка выше
3. **Рассчитать профессиональные очки навыков** (основные характеристики x множитель)
4. **Распределить профессиональные очки** по навыкам профессии
5. **Рассчитать очки личных интересов** (ИНТ x10)
6. **Распределить очки личных интересов** по любым навыкам
7. **Рассчитать производные характеристики** (ХТ, ОМ, ПЕР, бу)
8. **Снарядить персонажа** подходящим для сеттинга снаряжением
