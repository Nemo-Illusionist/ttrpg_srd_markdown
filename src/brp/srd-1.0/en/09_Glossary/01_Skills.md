# Skills Reference

All 56 skills from the Basic Roleplaying SRD, listed alphabetically with base chances and descriptions.

## Skill Categories

- **Combat Skills** — direct fighting and weapon use
- **Communication Skills** — social interaction and persuasion
- **Manipulation Skills** — fine physical tasks and craft
- **Mental Skills** — knowledge, reasoning, and perception
- **Physical Skills** — athletic and bodily activities
- **Perception Skills** — awareness and detection

## Skill Improvement

Skills improve through **Experience Checks**:

1. After a successful skill use during a session, the player marks an experience check for that skill.
2. Between sessions, the player rolls D100 for each checked skill.
3. If the roll is **above** the current skill value, the skill increases by 1D6 points.
4. Skills above 100% can still increase but become harder to raise.

## Skill Modifiers

| Modifier | Effect |
|---|---|
| Easy (x2) | Double the skill chance for simple tasks |
| Difficult (/2) | Halve the skill chance for challenging tasks |
| Opposed Roll | Both participants roll; lower successful roll wins |
| Complementary Skill | Success on a related skill may grant a bonus |
