# Professions Reference

All 12 professions from the Basic Roleplaying SRD with complete skill lists, key characteristics, and descriptions.

## How Professions Work

A profession determines which skills a character can allocate **professional skill points** to during character creation. Professional skill points equal the sum of the profession's two primary characteristics, multiplied by a setting-dependent factor (typically x10 or x20).

Characters may also receive **personal interest skill points** (typically INT x10) to allocate freely to any skill.

## Character Creation Summary

1. **Roll characteristics** (STR, CON, SIZ, INT, POW, DEX, APP)
2. **Choose a profession** from the list above
3. **Calculate professional skill points** (primary characteristics x multiplier)
4. **Allocate professional points** to the profession's listed skills
5. **Calculate personal interest points** (INT x10)
6. **Allocate personal interest points** to any skills
7. **Calculate derived characteristics** (HP, PP, MOV, db)
8. **Equip the character** with appropriate gear for the setting
