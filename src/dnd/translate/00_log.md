# Терминологические решения — D&D

Лог спорных переводческих решений: **почему** выбран конкретный вариант, какие альтернативы отвергнуты.

Глоссарий (`*_Glossary/00_Glossary.md`) фиксирует **что** переводить как. Этот файл фиксирует **почему**.

---

### Opportunity Attack → Провоцированная атака

- **Решение:** Устоявшийся перевод из официальных русских изданий D&D
- **Отвергнуто:** "Атака по возможности" (калька), "Оппортунистическая атака" (ложный друг)

### Advantage / Disadvantage → Преимущество / Помеха

- **Решение:** Устоявшийся перевод из официальных русских изданий
- **Отвергнуто:** "Адвантаж/Дизадвантаж" (транслитерация), "Превосходство/Затруднение"

### Feature → Умение (класса) / Особенность (расы, предмета)

- **Решение:** Контекстно-зависимый перевод по типу источника
- **Class Feature** → Умение класса
- **Racial Feature** → Особенность расы
- **Отвергнуто:** "Фича" (жаргон), "Свойство" (слишком общее)

### Saving Throw → Спасбросок

- **Решение:** Устоявшийся в русскоязычном сообществе термин (одно слово)
- **Отвергнуто:** "Спасительный бросок" (калька), "Бросок спасения" (тяжеловесно)
