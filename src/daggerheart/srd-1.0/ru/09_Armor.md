# Броня

У каждой брони есть название, базовые Пороги Урона и Базовый Класс Брони. У некоторых видов брони также есть Свойство.

- **Базовый Класс Брони** определяет, сколько Ячеек Брони броня даёт своему владельцу до применения дополнительных бонусов для расчёта итогового Класса Брони. Класс Брони ИП не может превышать 12.
- **Базовые Пороги** определяют Большой и Критический Пороги Урона владельца до применения бонусов для расчёта итоговых Порогов Урона. При записи Порогов Урона в раздел «Урон и Здоровье» листа персонажа всегда прибавляй Уровень персонажа к этим значениям.
- **Свойство** брони — это особое правило, действующее, пока броня надета.

**Без брони** Базовый Класс Брони твоего персонажа равен 0, Большой порог равен его Уровню, а Критический порог — удвоенному Уровню.

## Снижение входящего урона

Когда ты получаешь урон, ты можешь отметить одну Ячейку Брони, чтобы снизить тяжесть урона на один порог (Критический → Большой, Большой → Малый, Малый → Ничего). Если Класс Брони твоего персонажа равен 0, ты не можешь отмечать Ячейки Брони. Если эффект временно увеличивает твой Класс Брони, количество доступных Ячеек Брони увеличивается на ту же величину; когда эффект заканчивается, эти Ячейки Брони также становятся недоступными.
