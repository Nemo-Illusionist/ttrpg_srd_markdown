# Правила нормализации Markdown (Markdown Normalization)

Фаза C скилла `/cleanup-artifacts`. Механическая чистка форматирования после восстановления структуры.

Выполняется ПОСЛЕ layout recovery (Фаза B). К этому моменту структура документа уже восстановлена.

## Форматирование

- Разорванные bold/italic (`**text` без закрывающего `**`) — исправить
- Разорванные bold-italic (`***text` или `**_text`) — исправить
- Артефакты OCR: лигатуры `ﬁ`→`fi`, `ﬂ`→`fl`, `ﬀ`→`ff`
- Soft hyphens (`\u00AD`) — удалить

## Пробелы и пустые строки

- Двойные/множественные пробелы → один
- Trailing whitespace — удалить
- Потерянные пустые строки между разделами — восстановить
- Избыточные пустые строки (3+ подряд) — сократить до 2

## Тире и кавычки

- `--` → `—` (em dash)
- ` - ` в середине предложения → ` — `
- Кавычки: прямые → типографские (если в документе используются типографские)

## Не трогать

- Dice notation: `3D6`, `D10`, `D100`, `1D8+1`
- Rule references и кросс-ссылки
- System terms и игровые термины
- Числа и формулы
- Содержание и смысл текста
- Структуру документа (это задача layout recovery)
