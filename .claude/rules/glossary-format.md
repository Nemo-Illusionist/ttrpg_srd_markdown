# Формат глоссария

Единый стандарт структуры глоссария для EN и RU. Используется в `/build-glossary` и `/translate-glossary`.

## Структура директории

- Путь: `src/{game}/{version}/{lang}/{NN}_Glossary/`
- `{NN}` = следующий номер после последнего файла/папки в `src/{game}/{version}/en/`
- Номер одинаковый для EN и RU

## Файлы

- `00_Glossary.md` — основные термины, сгруппированные по категориям
- `01_Spells.md`, `02_Monsters.md`, ... — справочные таблицы сущностей
- Нумерация файлов: `01`, `02`, `03`, ...

## Формат `00_Glossary.md`

### EN версия

Каждая категория — таблица:

```markdown
## Conditions

| Condition | Description |
|---|---|
| Blinded | Cannot see... |
```

Типичные категории (адаптировать под конкретную игру):
- Характеристики (Abilities/Attributes)
- Условия/состояния (Conditions)
- Типы урона (Damage Types)
- Классы (Classes)
- Расы/происхождения (Species/Ancestries)
- Навыки (Skills)
- Типы существ (Creature Types)
- Ключевые слова механик (Keywords)

### RU версия

Начинается с раздела "Редакционные принципы":

```markdown
# Глоссарий

## Редакционные принципы

- Устоявшиеся термины из официальных переводов сохраняются
- Новые термины переводятся по смыслу, оригинал указывается в скобках при первом упоминании
- Имена собственные транслитерируются, оригинал в скобках
- Единообразие: один EN термин = один RU перевод во всех файлах
- Игромеханические термины с заглавной буквы (Сила, Ловкость, Испуганный)
```

Каждая таблица — с колонкой "Оригинал" вторым столбцом:

```markdown
## Состояния

| Состояние | Оригинал | Описание |
|---|---|---|
| Ослеплённый | Blinded | Не может видеть... |
```

## Формат справочных таблиц (01+)

### EN версия

Колонки зависят от типа сущности:

**Заклинания:**
```markdown
| Spell | Level | School | Classes |
|---|---|---|---|
| Acid Splash | Cantrip | Evocation | Sorcerer, Wizard |
```

**Монстры:**
```markdown
| Monster | CR | Type | Size |
|---|---|---|---|
| Aboleth | 10 | Aberration | Large |
```

**Магические предметы:**
```markdown
| Item | Type | Rarity | Attunement |
|---|---|---|---|
| Amulet of Health | Wondrous Item | Rare | Yes |
```

Адаптировать колонки под конкретную игру.

### RU версия

- **Первая колонка** — RU имя
- **Вторая колонка** — EN оригинал (всегда "Оригинал")
- Остальные колонки переведены

```markdown
| Заклинание | Оригинал | Уровень | Школа | Классы |
|---|---|---|---|---|
| Брызги кислоты | Acid Splash | Заговор | Воплощение | Чародей, Волшебник |
```

## Сортировка

- EN: по алфавиту (A-Z) в каждой таблице
- RU: по **русскому** алфавиту (А-Я) в каждой таблице

## Числа

Числа без изменений (уровни, КО, размеры, стоимость).
