# Бой

## Совершение атаки

Когда вы совершаете атаку, вы бросаете к20 и добавляете модификатор.

### Атаки ближнего боя

| Тип атаки | Характеристика | Дистанция |
|---|---|---|
| Безоружный удар | Сила | 5 футов |
| Атака оружием | Сила | 5 футов |

### Дальнобойные атаки

Дальнобойную атаку можно совершить только по цели в пределах дистанции оружия.

Этот абзац лишний — его нет в EN. Должна быть ошибка парной проверки.
