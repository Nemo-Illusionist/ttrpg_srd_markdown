## pair-en.md ↔ pair-ru.md (парная проверка)

| Проверка | Severity | Ошибка |
|---|---|---|
| Blockquotes | error | EN: 3 строки blockquote, RU: 2 строки (потеряна строка "It can fly at great speeds") |
| Списки | error | EN: 3 элемента, RU: 4 элемента (лишний "Четвёртый элемент") |
| Строки | warning | RU имеет лишний абзац ("Этот абзац лишний...") — количество строк отличается |

**Заголовки и таблицы** — парная проверка должна пройти без ошибок (структура совпадает).
